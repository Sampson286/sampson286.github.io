java正则使用及总结

  • java

正则表达是只存在我的大学记忆中了，最近几天在工作中遇到了切割字符串的需求。开始想了一些方式发现都不是很好决定用正则试一试。正好借此机会总结一下。

正则表达式，又称正规表示法、常规表示法（英语：Regular Expression，在代码中常简写为regex、regexp或RE），计算机科学的一个概念。正则表达式使用单个字符串来描述、匹配一系列符合某个句法规则的字符串。在很多文本编辑器里，正则表达式通常被用来检索、替换那些符合某个模式的文本。

  1. 这次主要需要分割的格式为:数字、汉字、字母为顺序的字符串如00晴Sunny01多云Cloudy，进行分割把数字取出作为键，其余的值以实例方式存为value。 
public class WeatherId2Detail {
/**为天气编码的字符串*/
private static String weather_detail="00晴Sunny01多云Cloudy02阴Overcast03阵雨Shower04雷阵雨Thundershower" +
        "05雷阵雨伴有冰雹Thundershowerwithhail06雨夹雪Sleet07小雨Lightrain08中雨Moderaterain09大雨Heavyrain10暴雨Storm" +
        "11大暴雨Heavystorm12特大暴雨Severestorm13阵雪Snowflurry14小雪Lightsnow15中雪Moderatesnow16大雪Heavysnow17暴雪Snowstorm" +
        "18雾Foggy19冻雨Icerain20沙尘暴Duststorm21小到中雨Lighttomoderaterain22中到大雨Moderatetoheavyrain23大到暴雨Heavyraintostorm" +
        "24暴雨到大暴雨Stormtoheavystorm25大暴雨到特大暴雨Heavytoseverestorm26小到中雪Lighttomoderatesnow27中到大雪Moderatetoheavysnow" +
        "28大到暴雪Heavysnowtosnowstorm29浮尘Dust30扬沙Sand31强沙尘暴Sandstorm53霾Haze99无Unknown";
public static HashMap<String, WeatherDetailInfo> weatherDetail=new HashMap<>();
static{
    int i=0;
    Pattern pattern = Pattern.compile("[\\u4e00-\\u9fa5]+|\\d+|[a-z|A-Z]+");
    Matcher matcher = pattern.matcher( weather_detail );
    String key = "" ;
    String v1 = "";
    String v2="";
    while ( matcher.find() ) {
        i++;
        if(i==1){
            key=matcher.group();
        }else if(i==2){
            v1=matcher.group();
        }else if(i==3){
            v2=matcher.group();
            i=0;
            weatherDetail.put(key, new WeatherDetailInfo(v2, v1));
        }
    }
}}

在此方法中主要使用到知识有：

  1. \u4e00-\u9fa5为Unicode编码中的汉字编码
  2. 正则的元字符，+匹配前面的子表达式一次或多次(大于等于1次）
