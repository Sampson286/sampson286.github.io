Retrofit2 源码学习

  • android


Evernote Export









最近在工作中了解到网络请求框架Retrofit，通过官方的引导文档，先了解一下Retrofit的使用，再跟深入的学习。
  1. Retrofit的使用
  • 首先需要创建一个用于表示Http请求的接口
public interface GitHubService {
@GET("users/{user}/repos")//执行资源的地址后缀
Call<List<User>> listRepos(@Path("user") String user);
}
  • 再接着需要通过Retrofit类来创建一个实现了GitHubService接口的代理对象
Retrofit retrofit = new Retrofit.Builder()

.baseUrl("https://api.github.com/&quot;)
.build();
GitHubService service = retrofit.create(GitHubService.class);
  • 得到请求服务的请求对象
Call<List<User>> call=service.listRepos("zhaoyongchao");//得到请求对象
call.execute();//同步请求
//发起异步请求
call.enqueue(new Callback<List<User>>() {
@Override
public void onResponse(Call<List<User>> call, Response<List<User>> response) {
Log.i(TAG,response.body().get(0).getName());
}
@Override
public void onFailure(Call<List<User>> call, Throwable t) {
Log.i(TAG,"获取失败onFailure"+t.getMessage());
}
});
